## 三角形

```css
div{
    width: 0;
    height: 0;
    border: 100px solid transparent;
    border-top: 100px solid red;
}
```



## 扇形

```css
div{
    width: 0;
    height: 0;
    border: 100px solid transparent;
    border-radius: 100px;
    border-top-color: red;
}
```



## 圆形

```css
div{
    width: 0;
    height: 0;
    border: 100px solid red;
    border-radius: 100px;
}
```



## 自适应矩形

```css
.div{
    width: 100%;
    height: 0;
    padding-top: 75%;
    background-color: red;
}
```
